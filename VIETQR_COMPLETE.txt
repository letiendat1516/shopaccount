### 3. **API Routes Implemented**
```
POST   /checkout/vietqr
       - Creates payment snapshot
       - Generates QR code
       - Starts polling
       - Response: { success, snapshotId, qrUrl, amount, ... }

GET    /checkout/vietqr/verify/:snapshotId
       - Checks payment status
       - Completes order if paid
       - Used by polling mechanism

POST   /checkout/vietqr/webhook
       - Receives bank notifications
       - Validates webhook
       - Processes payment automatically
       - Optional: requires bank API integration

GET    /api/vietqr/config
       - Returns VietQR config for admin panel
```

### 5. **Automatic Payment Verification System**

**Option A: Polling (Default)**
- Frontend polls `/checkout/vietqr/verify` every 1 second
- Runs for maximum 5 minutes (300 attempts)
- User sees real-time status updates
- Manual "Verify Payment" button for on-demand check

**Option B: Webhook (Recommended for Production)**
- Bank sends notification to `/checkout/vietqr/webhook`
- Server processes immediately
- Instant payment confirmation
- Requires bank/third-party API integration

**Option C: Manual**
- User clicks "Verify Payment" button
- Single-check verification
- Useful if customer needs to verify later

### 6. **Order Completion Flow**
When payment is confirmed, system automatically:
```
✅ Creates Payment record with all details
✅ Adds products to user.ownedProducts
✅ Generates and sends invoice PDF
✅ Sends confirmation email (if enabled)
✅ Assigns Discord roles (if applicable)
✅ Updates store statistics
✅ Clears user cart
✅ Logs payment to Discord
✅ Redirects to invoice page
```

## Common Questions

### Q: How long does payment take to confirm?
**A:** 1-5 minutes automatically via polling. Instant if using bank webhook.

### Q: Does it work with all banks?
**A:** Yes! VietQR standard works with 50+ Vietnamese banks.

### Q: Can customers see their transaction?
**A:** Yes, they get email invoice and can view at `/invoice/:transactionId`

### Q: What if payment fails?
**A:** Cart is marked as `expired` after 5 minutes. Customer can retry.

### Q: How do I integrate with bank API?
**A:** Update `webhookUrl` in settings to your bank's webhook endpoint.

### Q: Is it secure?
**A:** Yes - CSRF protected, signature validation, amount verification, IP logging.
